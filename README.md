A Google Apps Script-based automation tool developed to optimize budget data transfer across Google Sheets through a dynamic key-to-header mapping engine. The script intelligently reads source data, aligns it with destination schemas, and appends fully structured records while capturing critical identifiers such as Budget Name and Opportunity ID. It also includes a custom UI menu for one-click execution, making it accessible for non-technical users. By eliminating manual data entry and ensuring consistent formatting, this solution improved data accuracy, standardized financial workflows, and enabled scalable, repeatable processing of large datasets.
